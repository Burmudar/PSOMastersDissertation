--- The <#840#>HLR<#840#> is a database that permanently stores information pertaining to a given set of subscribers. It needs to store a wide range of subscriber parameters because it is the reference source for anything <#841#>gsm<#841#> subscriber related in the network[#GSMSysEngin##1###].

Subscriber parameters that are stored in the database include billing information, routing information, identification numbers, authentication parameters and subscribed services[#GSMSysEngin##1###]. The following information is also stored, but the information is of a temporary nature and can change at any time: The current <#844#>VLR<#844#>and <#845#>MSC<#845#> the subscriber is registered with and whether the subscriber is roaming [#GSMSysEngin##1###].
