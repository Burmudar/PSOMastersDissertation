The term NP stems from the field known as complexity analysis. Algorithms are typically measured for their worst running time using #tex2html_wrap_inline14934# mathend000# notation. The field of complexity analysis is more interested in measuring complexity of a problem than the running time of an algorithm[#AIModernApproach##1###].

The field of complexity analysis makes a keen distinction between problems that can be solved by algorithms in polynomial time and problems that cannot be solved in polynomial time using any available algorithm[#AIModernApproach##1###]. An algorithm is said to be of polynomial time if the number of steps it needs to perform to solve the problem is given by #tex2html_wrap_inline14936# mathend000# where #tex2html_wrap_inline14938# mathend000# is a nonnegative integer and #tex2html_wrap_inline14940# mathend000# refers to the complexity of the input[#AIModernApproach##1###].

A distinction needs to be made between finding a possible solution and determining whether a result is a valid solution to an NP problem[#AIModernApproach##1###]. Verifying whether some result is indeed a solution to an NP problem is a quick operation[#AIModernApproach##1###]. Finding a solution through the use of an algorithm is what polynomial time refers to.

Algorithms that are able to solve problems in polynomial time have worst case running times of #math49##tex2html_wrap_inline14942# mathend000# and #tex2html_wrap_inline14944# mathend000#[#AIModernApproach##1###]. These classes of problems are easy to solve using modern algorithms available today. Problems that can be solved with these kind of running times are classified as being in the P range of complexity problems[#AIModernApproach##1###].

Another range of problems consists mostly of problems that cannot be solved in polynomial time[#AIModernApproach##1###]. Hence, if an algorithm were to try each and every possible solution, it would take an arbitrarily long time, which cannot be determined, which is why these problems also have the characteristic of being non-deterministic[#AIModernApproach##1###]. Problems in this range are referred to as being in the NP range of complexity problems[#AIModernApproach##1###].

There is another range of NP problems that are a subset of NP problems, which are referred to as the ``most extreme'' problems within NP. These problems are collectively known as NP-Complete problems and are the most difficult problems to determine feasible solutions for in the NP problem range[#AIModernApproach##1###].
