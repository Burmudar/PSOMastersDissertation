Network databases

The <#835#>HLR<#835#>, <#836#>AUC<#836#> and <#837#>EIR<#837#> are the three `back-end' databases which store and provide information for the rest of the <#838#>gsm<#838#> network. In this subsection each one of the databases that form part of the `back-end' is discussed briefly and a description is given of the core functions that each database performs in the network.
