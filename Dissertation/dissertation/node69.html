Metaheuristics Algorithms

NP-Complete4.1 problems have been proven to not be solvable in polynomial time by traditional uninformed search algorithms such as Breath First Search and Depth-First Search[#AIModernApproach##1###].

Uninformed algorithms are not able to distinguish whether a particular goal state is more ``correct'' than any other goal state[#AIModernApproach##1###].

Uninformed search algorithms search spaces are represented by a tree structure[#AIModernApproach##1###]. A search is performed in the search space by starting at a root node, the algorithm expands successive nodes based on a strategy[#AIModernApproach##1###]. Bread-first search continiously expands all successive nodes and Depth-First search expands the deepest node first[#AIModernApproach##1###].

The path taken to a final goal state node represents a solution[#AIModernApproach##1###]. In a worst-case scenario, an uninformed search will expand every single node in its search space, thus each and every single possible solution is evaluated[#AIModernApproach##1###].

It is not always viable to test every possible solution in a given problem search space, especially in NP-Complete problems, since their search spaces are huge or infinite. This is why traditional algorithms are not able to produce optimal solutions in polynomial time[#AIModernApproach##1###].

Metaheuristic algorithms are considered to be general-purpose algorithms and can thus be applied to a wide variety of optimisation problems with only small modifications that need to made to the algorithm model[#MetaGraph##1###].

Metaheuristic algorithms do not dictate all aspects of the search procedure but define guidelines and aspects which directs the search[#HandbookofMH##1###]. Therefore a metaheuristic can be defined as follows: the algorithm selects candidate solutions from a neighbourhood with respect to one or more current solutions of which the candidates are either rejected or accepted[#HandbookofMH##1###].

As can be gathered from the name, a metaheuristic algorithm uses some sort of heuristic. A heuristic is a decision rule. Algorithms use heuristics to make decisions by applying the heuristic to the data the algorithm is currently working on[#AIModernApproach##1###,#NatureInspiredMetaHeuristic##1###].

When an algorithm utilises a heuristic its forms part of a group known as informred search algorithms. As the name implies the algorithm is more ``informed'' using the heuristic the algorithm gains additional information about its search space. With this additional information the algorithm is able to performa a better search for a possible solution[#AIModernApproach##1###].

Heuristics are strategies that direct the search based on the information available, to move towards areas where there is a higher probability of obtaining high quality candidate solutions [#AIModernApproach##1###]. A heuristic is able to dictate what an algorithm must do for its next iteration by evaluating the current internal state of the algorithm, i.e. whether it should move to a different point in the search space, generate new data, or select the current data as the most optimal solution[#AIModernApproach##1###,#NatureInspiredMetaHeuristic##1###].

In general ``meta'' means beyond or higher level[#AIModernApproach##1###,#NatureInspiredMetaHeuristic##1###]. A metaheuristic therefore refers to a heuristic that is more complex with regard to the decisions it is able to make compared with a standard heuristic[#AIModernApproach##1###,#NatureInspiredMetaHeuristic##1###]. With a standard heuristic only the current state is considered[#AIModernApproach##1###]. A metaheuristic takes additional information into account when considering a current state[#NatureInspiredMetaHeuristic##1###]. This additional information can be previous states, or states that are considering to be close in the search space[#AIModernApproach##1###,#NatureInspiredMetaHeuristic##1###].

Metaheuristic-based search algorithms are not guaranteed to find the most optimal solutions in the search space; instead these algorithms are used to find near-optimal solutions. Thus most algorithmic development in the metaheuristic domain focuses on developing new techniques that will increase the probability that a good solution will be obtained in difficult combinatorial problems [#MetaAgricultural##1###].

Similarly, metaheuristics are not guaranteed to find suitable solutions or perform well in each problem domain it is applied to. The quality of the solution and performance of the metaheuristic is very much dependent upon on the expertise of the algorithm designer [#AutoComplexMeta##1###]. Besides the algoritm designer modifying the algorithm; metaheuristic algorithms that are inherently population-based, hybrid and/or distributed, use the concept of social and self-organisation to better exploit the solution space[#Self-AdaptiveMeta##1###].
