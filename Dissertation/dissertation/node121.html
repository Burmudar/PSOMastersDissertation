Building a solution

--- The <#4085#>ACO<#4085#> builds an optimal solution. Therefore, early decisions made by the ants still influence the plan later for better or for worse. A good decision might seem to be good early on, but later the algorithm might be better off with a slightly worse decision. In the <#4087#>FAP<#4087#>, a cell can have multiple interfering cells, but a particular ant only knows about one link between two cells and not about the other interfering cells. Thus an ant will find the optimal path on the first interfering link between two cells, in other words it will optimise the channels allocated to these cells so that interference is low. The first interfering link is now optimised, and subsequent ants will reinforce this channel allocation since the interference is low. When the ants later reach the other cells that also interfere with the first cell that has been optimised, they will have difficulty changing the assignments that have already been made, since the pheromone representing that assignment is too strong to disregard.

The above disadvantages have only been identified by critically evaluating the algorithm as a possible point of interest to produce an optimal solution for the <#4088#>FAP<#4088#> for this dissertation. Even with these disadvantages the <#4089#>ACO<#4089#> has achieved success in producing high quality optimal solutions for the <#4090#>FAP<#4090#>.

In research conducted by Luna et al.[#ACOvsEA##1###] an <#4092#>ACO<#4092#> algorithm was applied to a custom cellular network instance. This network instance had 711 sectors with 2 612 transceivers, which needed to be assigned frequencies. For their particular network, only 18 channels were available for assignment. The channels started at 134 and ended at 151[#ACOvsEA##1###].

The authors presented two versions of the algorithm. The first version used no heuristic information (henceforth referred to as <#4094#>ACO<#4094#>*) and the other version used heuristic information to update the pheromone laid done by the artificial ants[#ACOvsEA##1###].

With regard to the heuristic updating of the pheromone trails, the authors opted to increase the pheromone by some magnitude[#ACOvsEA##1###]. This magnitude was handtuned to be 100. The heuristic only increases a certain path's pheromone if the frequencies assigned to the transceivers represented by this path differ enough so as to not cause significant interference[#ACOvsEA##1###]. Thus, the heuristic aims to amplify good choices made previously by the algorithm for the next iteration of the algorithm.

By evaluating the results obtained, one can clearly see that the <#4100#>ACO<#4100#> version that incorporates heuristic information to reinforce pheromone trails outperforms the version that does not [#ACOvsEA##1###].
