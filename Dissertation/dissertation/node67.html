Introduction

Metaheuristics is a subdomain of the <#2421#>AI<#2421#> domain[#AIModernApproach##1###]. It evolved out of a need for more efficient search techniques with regard to hard problems.

Metaheuristics forms part of a collective body of algorithms that use heuristics to search a particular domain's search space for the most optimal solution[#AIModernApproach##1###,#NatureInspiredMetaHeuristic##1###]. Some problems explicitly define constraints to which the produced solution must adhere to. Problems that define such constraints are referred to as constraint optimisation problems[#FundamentalSwarm##1###]. Problems where there are either no constraints or just a boundary constraint defined are referred to as unconstraint problems[#FundamentalSwarm##1###].

Constraints can be either hard or soft. A hard constraint is defined as a certain condition an algorithm or potential solution is not allowed to violate[#AIModernApproach##1###,#NatureInspiredMetaHeuristic##1###,#Karen2004##1###,#Eisenblatter##1###]. A soft constraint is allowed to be violated but there is some sort of penalty or cost involved which is imposed onto the potential solution, which lowers its desirability[#AIModernApproach##1###,#NatureInspiredMetaHeuristic##1###,#Karen2004##1###,#Eisenblatter##1###].

An optimal solution would therefore be any solution that violates no hard constraints and violates no or a minimum number of soft constraints[#AIModernApproach##1###,#NatureInspiredMetaHeuristic##1###,#Karen2004##1###,#Eisenblatter##1###]. Algorithms that are classified as being part of the collective body of algorithms known as metaheuristic algorithms are Tabu Search[#TabuVechicleRoutingWithTimeWindows##1###,#TabuCSP##1###], Simulated Annealing [#SASingleMultiObj##1###,#CurveFittingSA##1###] and Genetic Algorithm[#GATSP##1###,#GeostatisticalGA##1###].

The above-mentioned algorithms are not the only algorithms to form part of this subdomain, but they are the algorithms that have received the most attention in the literature and produce good results[#SweepMeta##1###].

The main focus of this chapter is on each of the above algorithms. Before each of the algorithms is discussed, a brief overview is given of the various characteristics that metaheuristic algorithms exhibit.

<#2433#>AI<#2433#> search algorithms operate in search spaces, where they occupy various states while they are searching for a solution or goal state in a search space. Before these algorithms can be properly introduced, the concept of search spaces and states needs to be introduced.
